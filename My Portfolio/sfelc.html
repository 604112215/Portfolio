SFELC WEBDESIGN

SFELC is the first version of the website for San Francisco Engineering Leadership Community (SFELC), a community for engineering leaders in Silicon Valley. It recreates education to improve soft skills( e.g., communication, self-awareness, etc.) for engineers. We now have 1500+ leaders(includes CTO, VP) from over 700 companies.

My Role

Design UI/UX independently for SFELC website. Sign in/up the flow, RSVP flow, Book List, Comment feature, etc. , and deliver spec files to engineer. Create and maintain design guidelines and libraries which could be used by the whole design and engineering team. I am also working on building up other essential functions including Training Program, Job Board, Peer Group, etc.

SIGN IN/ UP

Users can easily log in our website either with LinkedIn account or Email. After the first login, there is a questionary that needed be filled out.

RSVP

RSVP shows all the information about the upcoming event. Users can buy an event ticket here.

Two types of event

Free Event
1. If there are available spots, the user can directly RSVP into the upcoming event and get a follow-up confirmation email.
2. If the event is free but doesn’t have available spots left, the user can Join Wait List. Once there are any open spots, they will enroll in the event automatically and get follow-up confirmation. (Join Wait List only available for a free event)

Charged Event
1. If there are available spots, the user needs to pay the event fee and then RSVP into the upcoming event and get a follow-up confirmation email.

BOOK LIST

Book List is a very crucial part to the SFELC. It is a place where people exchange value and find common interests together. The Book List is composed of three parts:
1.Recommend a book which allows users to recommend books to our community and fellows.
2.Comment Section which let members discuss books.
3. Like which enable members to mark which books and comments do they like.

COMMENT SECTION
